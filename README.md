## Getting Started

This is an example of how you may give instructions on setting up your project locally.
To get a local copy up and running follow these simple example steps.

### Prerequisites

* Flutter and Dart. You will need Flutter version 3.7 or later. https://flutter.dev/docs/get-started/install
  
* Docker. Docker is used to manage Postgres and (optionally) Redis. https://docs.docker.com/get-docker/

* Serverpod
  ```sh
  dart pub global activate serverpod_cli
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/flutterindiadev/dart_backend.git
   ```
2. Install Flutter and Dart packages
   ```sh
   flutter pub get
   ```
